## 🧰 Troubleshooting

### The app will not open
- Check that the file finished downloading.
- Make sure you extracted the zip folder.
- Try running it as an administrator.

### The app cannot connect to Keitaro
- Check the tracker URL.
- Check the API key.
- Make sure your internet connection works.
- Confirm that your Keitaro account is online.

### No data appears
- Refresh the view.
- Check your date range.
- Confirm that the campaign has traffic.
- Make sure the right account is selected.

### The app shows an error with commands
- Try a shorter request.
- Use plain words.
- Avoid very long messages.
- Check that your account is connected
